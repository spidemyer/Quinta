# Diferença entre Biblioteca e Framework

**Biblioteca:** É um conjunto de funções que nós chamamos quando precisamos,
    temos o controle total de como usar.
*Exemplo: Math, fs, prompt-sync.*

**Framework:** É uma estrutura pronta que dita como deve ser usada, chama o código nos momentos certos e a gente que segue as regras.
*Exemplo: Express.js.*
